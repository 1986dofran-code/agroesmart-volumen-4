# AgroSmart Rural - Volumen 4

Proyecto web base para registrar y revisar datos diarios de produccion en la Finca La Esperanza.

## Descripcion

AgroSmart Rural permite llevar un control sencillo de:

- Litros de leche producidos.
- Kilos de maiz cosechados.
- Animales en produccion.
- Observaciones diarias.
- Indicadores principales y alertas de registros con posible riesgo.

## Estructura del proyecto

```text
sistema-estudiante/
  index.html
  css/
    styles.css
  js/
    app.js
```

## Como usarlo

1. Abre la carpeta `sistema-estudiante`.
2. Ejecuta el archivo `index.html` en el navegador.
3. Registra los datos diarios o usa el boton `Cargar demo` para ver informacion de prueba.

## Funciones principales

- Registro diario de produccion.
- Tabla de registros.
- Calculo de totales y promedio de leche.
- Conteo de alertas.
- Cambio de tema visual.
- Eliminacion de registros.
